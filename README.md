## Submitool
Automated commiting and taging tool to push git repository to remote faster.
### version 2.0

## Installation

Clone the repository

```bash
git clone https://github.com/gaetanslrt/submitool.git
cd submitool
```

Run the installer

```bash
./install.sh
```

## Usage

Inside a git repository run
```bash
submitool
```
